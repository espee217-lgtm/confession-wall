# Venom Screen Record Sprite Sheet

This patch contains only new asset files, not the full project.

Generated from the uploaded MP4 screen recording.

## Main files

- `client/src/assets/avatarFrames/venom-screen-record-spritesheet.png`
- `client/src/assets/avatarFrames/venom-screen-record-preview.gif`
- `client/src/assets/avatarFrames/venom-screen-record-frame.css`
- `client/src/assets/avatarFrames/venom-screen-record-frames/`

## Sprite sheet details

- Frames: 48
- Layout: 8 columns x 6 rows
- Frame size: 258px x 258px
- Sheet size: 2064px x 1548px
- Background: transparent PNG
- Source video: 310px x 282px, 30.00 FPS, 148 frames

## Note

This is pixel-faithful to the screen recording frames. A screen recording may still contain MP4 compression/antialiasing from the source.
No npm install is required.
